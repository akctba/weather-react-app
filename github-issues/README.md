# GitHub Issues to Create

This directory contains individual issue templates that should be created as GitHub issues.

Each file represents one issue and includes:
- Issue title
- Labels to apply
- Detailed description
- Code examples
- Acceptance criteria

## How to Create Issues

Since automated issue creation is not available, please manually create each issue using the GitHub web interface:

1. Go to https://github.com/akctba/weather-react-app/issues/new
2. Copy the title from each file
3. Copy the body content
4. Apply the suggested labels
5. Submit the issue
